# SimpleCalculator

This project was implemented to execute basic mathematical operations.

## Development

1.Maven-3.6.3
2.Spring Boot 2.4.1
3.H2 Database  
4.Junit 

## Running

Run it as Java Application. Navigate to `http://localhost:8080/`. The app will automatically, attached screen shot for the reference.
